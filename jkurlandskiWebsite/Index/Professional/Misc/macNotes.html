Finder => Command Line

If you're in Finder and you want to get to a command prompt at that directory, here's what you do.

In System Preferences choose Keyboard and then Shortcuts. From the left side nav, choose Services. Then from the main area under Files and Folders, choose New Terminal at Folder and/or New Terminal Tab at Folder.

Now you can right mouse click or control click on a folder and choose Services > New Terminal at Folder.

Command Line => Finder

Opening Finder at a particular directory is even easier. At the command line, do a pwd. Then highlight the path and copy it into your clipboard.

Now open Finder and hit Cmd-Shift-G. Paste the contents of your clipboard into the text field and--voila!

TextMate

After years of using Mac's TextEdit for text editing, and resorting to Xcode whenever the former's regex-search abilities were too limited, I finally discovered and installed TextMate. One thing I really like about it is the ability, once you perform the right steps, to open a text file with TextMate at the command line. See this linke:

TextMate Shell Utility

Once you've done the set up, simply type "mate " at the command line.

File diff app for Mac

Mac's Xcode developer tools come with a pretty darn cool visual diff tool called FileMerge. The only thing is, the application is a little hard to find. Assuming you've already installed Xcode to your Applications folder, you can locate FileMerge with Finder by going to Applications and right-clicking (or Command-Clicking) on Xcode and selecting "Show Package Contents." Then navigate to Contents/Applications. If you think you'll make frequent use of FileMerge, you can now drag it to the Dock at the bottom of your screen.
