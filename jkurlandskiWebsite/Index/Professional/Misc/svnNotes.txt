Jerrys-MacBook-Pro:jkurlandski jkurlandski$ svn status

(Reversion successful, but we still haven't added the new files. We could do each one separately, e.g. with "svn add jkurlandskiWebsite/Index/Professional/toDo.txt"; we could do both together with "svn add * --force," but this would again add the ignored files we've just reverted; instead we'll do them both with a Unix shell command.)

Jerrys-MacBook-Pro:jkurlandski jkurlandski$ for i in $(svn status | grep \? | awk '{print $2}'); do svn add $i; done
