SVN Tasks

JERRY: Red bar at top needs to point to this page's subtopic. Change filename to htmlTasks.html

This page shows you how to link individual SVN commands together in order to accomplish a particular task.

The SVN command line can be a little intimidating at first, but I find it becomes less so when you see the commands not in isolation but as part of a series of steps with a single aim in mind. If the individual commands are like words, then this page shows you how to string these words together to form a coherent sentence.

This page will probably make sense even if you're not yet familiar with the SVN commands being used. In either case, for more information on each command you can refer to the internal link on SVN Commands in the sidebar.

Task

> Update your working directory so that it has the latest version in the repository, leaving local changes untouched.

> Then commit local changes to the repository.


First check the status to see if there are any conflicts. (JERRY: conflicts are marked with )

$ svn status
?       Misc/svnExamples.html
M       toDo.txt

$ svn status -u
        *       74   Misc/svnCheatSheet.html
?                    Misc/svnExamples.html
        *       74   Misc/svnNotes.txt
M                    toDo.txt
Status against revision:     77

Locally, toDo.txt has been modified, and svnExamples.html is new to the file system but not added to the repository. On the repository side, the repository is at Revision 77, but it contains two files from Revision 74 that have not been copied locally.

The status checks reveal no conflicts--perform the update. Then check status again.

$ svn update
Updating '.':
U    Misc/svnCheatSheet.html
U    Misc/svnNotes.txt
Updated to revision 77.

$ svn status -u
?                    Misc/svnExamples.html
M                    toDo.txt
Status against revision:     77

The local changes aren't ready to be committed because you haven't explicity stated you want the new file to be added to the repository. Do so now.

$ svn add Misc/svnExamples.html
A                   Misc/svnExamples.html

$ svn status
A                    Misc/svnExamples.html
M                    toDo.txt
Status against revision:     77

Finally you're ready to commit.

$ svn commit -m "New file svnExamples.html. Modified toDo.txt."
Adding         Misc/svnExamples.html
Sending        toDo.txt
Transmitting file data .
Committed revision 78.

Task

> Move one file from a subdirectory to another directory.

> Remove all other files in the original subdirectory from the repository as well as from your working copy.


First perform the move and the delete commands.

$ svn status

$ ls Databases
DatabaseNotes.txt   databaseTemplate.html   mySqlCheatSheet.html    mysqlCommandLine.txt

$ svn move Databases/mySqlCheatSheet.html Misc
A         Misc/mySqlCheatSheet.html
D         Databases/mySqlCheatSheet.html

$ svn delete Databases/DatabaseNotes.txt
D         Databases/DatabaseNotes.txt

$ ls Databases
databaseTemplate.html   mysqlCommandLine.txt

$ svn delete Databases/mysqlCommandLine.txt
D         Databases/mysqlCommandLine.txt

$ svn delete Databases/databaseTemplate.html
D         Databases/databaseTemplate.html

$ svn delete Databases
D         Databases

$ svn status
D       Databases
D       Databases/DatabaseNotes.txt
D       Databases/databaseTemplate.html
D       Databases/mySqlCheatSheet.html
D       Databases/mysqlCommandLine.txt
A  +    Misc/mySqlCheatSheet.html
M       Misc/svnExamples.html

Don't forget to commit your changes, thus.

Deleting       Databases
Adding         Misc/mySqlCheatSheet.html
Sending        Misc/svnExamples.html
Sending        toDo.txt
Transmitting file data ..
Committed revision 138.
