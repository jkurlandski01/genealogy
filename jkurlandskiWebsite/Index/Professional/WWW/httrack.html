Listing Starting Httrack

    Navigate to the directory where you want to extract to.
    $ httrack http://www.genericradio.com -O /tmp/httrack -Y
    or
    $ httrack http://www.simplyscripts.com/ -O ~/toDelete/httrack +http://www.simplyscripts.com/ +*.pdf +*.doc -http://www.amazon.com/*.html -http://www.simplyscripts.com/2013/*
    or
    $ httrack http://www.imsdb.com -O ~/toDelete/httrack/seinfeld +http://www.imsdb.com/*.html -http://www.imsdb.com/TV/*.html -http://www.imsdb.com/TV%20Transcripts/*.html

Stopping a running HTTRack process

    $ ctrl-Z
    $ ps -ef | grep httrack
    $ kill -9

Listing Continuing a halted operation

    Navigate to the directory where you've been running httrack.
    $ httrack -i
