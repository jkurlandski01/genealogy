Converting Collections: Arrays to Lists, Sets to Arrays and Lists

The code on this page is pretty basic, but sometimes I forget exactly how to convert one type of Java collection to another. I wrote this page to serve as a quick reference, mostly for myself.

Converting a Set to a List

Converting a Java Set to a List is so easy it's hardly worth discussing. But I thought I might as well be complete.

Listing 1: Converting a Java Set to a List

    // With Guava.
    Set<String> set1 = Sets.newHashSet("London", "Tokyo", "Warsaw"));
    List<String> list1 = Lists.newArrayList(set2);
    
    
    // In case Guava isn't available.
    Set<String> set2 = new HashSet<String>();
    set2.add("London");
    set2.add("Tokyo");
        
    List<String> list2 = new ArrayList<String>(set2);

Converting a Set to an array

Converting a Java Set to an array has a tricky syntax that I can never remember. Note that my example here assumes you are using Guava.

Listing 2: Converting a Java Set to an array

    Set<String> set1 = Sets.newHashSet("London", "Tokyo", "Warsaw"));
    String[] keysArray = set1.toArray((new String[set1.size()]));

Converting an array to a List

Listing 3: Converting an array to a Java List

    String[] keysArray = {"Prague", "Minsk", "Barcelona"};

    List keysList = Arrays.asList(keysArray);
