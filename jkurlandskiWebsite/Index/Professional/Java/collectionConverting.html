Converting Java Collections

JERRY: start with summary ?

The code on this page is pretty basic, but sometimes I forget exactly how to convert one type of collection to another--especially when it comes to arrays. I wrote this page to serve as a quick reference, mostly for myself.

Converting an array to a List

Converting an array to a Java List uses the handy Arrays.asList( ) method. In this as well as the rest of my examples, I use a unit test to illustrate and verify the functionality.

Listing 1: Converting an array to a Java List

    String[] cityArray = {"Prague", "Minsk", "Barcelona"};
    List<String> cityList = Arrays.asList(cityArray);
    
    assertEquals(3, cityList.size());

Converting a Set to an array

Converting a Java Set to an array has a tricky syntax that I can never remember. Note that my example here assumes you are using Guava.

Listing 2: Converting a Java Set to an array

    Set<String> set1 = Sets.newHashSet("London", "Tokyo", "Warsaw"));
    String[] keysArray = set1.toArray((new String[set1.size()]));

Converting a Set to a List and a List to a Set

Converting a Java Set to a List and vice versa are so easy they're hardly worth mentioning. But I thought I might as well be complete.

Listing JERRY: Converting a Java Set to a List

    // With Guava.
    Set<String> set1 = Sets.newHashSet("London", "Tokyo", "Warsaw");
    List<String> list1 = Lists.newArrayList(set1);
    assertEquals(3, list1.size());
    
    
    // In case Guava isn't available. Under Java 1.8.
    Set<String> set2 = new HashSet<>();
    set2.add("London");
    set2.add("Tokyo");
    set2.add("Warsaw");
        
    List<String> list2 = new ArrayList<>(set2);
    assertEquals(3, list2.size());
