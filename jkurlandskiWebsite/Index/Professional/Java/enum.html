Java Enums

I find that the enum type is one of the most useful tools Java has to offer. Whenever you have a class whose most important property is some kind of String, you should ask yourself if an enum wouldn't be a better implementation. If only a very limited number of String variables are valid for the property, then an enum is probably the better option. JERRY: examples: person last name is bad; parts of speech is good

Listing X: Creating a simple enum

public enum PartOfSpeech {
	NOUN, VERB, ADJECTIVE, ADVERB
}

Listing X: Using valueOf to decide if a string matches an enum type

// Note that this test is case-insensitive.
public static boolean isPartOfSpeech(String input)    {
	try {
			PartOfSpeech role = PartOfSpeech.valueOf(input.toUpperCase());
	} catch (IllegalArgumentException e) {
		// The relation isn't in our enum.
		return false;
	}
	return true;
}
