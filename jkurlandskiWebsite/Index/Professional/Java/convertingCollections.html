Converting Java Collections

The code on this page is pretty basic, but sometimes I forget exactly how to convert one type of collection to another—especially when it comes to arrays. I've written this page to serve as a quick reference—for myself or for anyone else who may find it useful.

In all of my examples, I use JUnit's assertEquals( ) to verify the functionality. Also, I'm instantiating my Set and List data structures with Guava. See my separate "Intro to Guava" page if you are unfamiliar with this library, but you could just as well use vanilla Java instantiation (as in "new HashSet<String>()"). The point of this page—converting from one type of collection to another—doesn't depend on how you instantiate the Java collections.

Converting an array to a List

Converting an array to a Java List uses the handy Arrays.asList( ) method.

Listing 1: Converting an array to a Java List

    String[] cityArray = {"Prague", "Minsk", "Barcelona"};
    List<String> cityList = Arrays.asList(cityArray);
    
    assertEquals(3, cityList.size());

Converting an array to a Set

To convert an array to a Java Set, first you convert it to a List, then convert the List to a Set. In the listing below, I do the whole thing in one fell swoop.

Listing 2: Converting an array to a Java Set

    String[] cityArray = {"Prague", "Minsk", "Budapest"};
    Set<String> citySet = Sets.newHashSet(Arrays.asList(cityArray));
    
    assertEquals(3, citySet.size());

Converting a Set or List to an array

Converting a Java Set to an array has a tricky syntax that I can never remember.

Listing 3: Converting a Java Set to an array

    Set<String> set1 = Sets.newHashSet("London", "Beijing", "New Delhi");
    String[] array1 = set1.toArray((new String[set1.size()]));
    
    assertEquals(3, array1.length);

As the next listing shows, converting a List to an array is almost the same.

Listing 4: Converting a Java List to an array

    List<String> list1 = Lists.newArrayList("Amsterdam", "Chicago", "Mexico City");
    String[] array1 = list1.toArray((new String[list1.size()]));
    
    assertEquals(3, array1.length);

Converting a Set to a List and a List to a Set

Converting a Java Set to a List and vice versa are so easy they're hardly worth mentioning. But I thought I might as well be complete.

Listing 5: Converting a Java Set to a List

    Set<String> set1 = Sets.newHashSet("London", "Tokyo", "Warsaw");
    List<String> list1 = Lists.newArrayList(set1);
    
    assertEquals(3, list1.size());

We already saw the conversion of a List to a Set in Listing 2 above, but for those who missed that, here it is again, in a more obvious form.

Listing 6: Converting a Java List to a Set

    List<String> list1 = Lists.newArrayList("London", "Tokyo", "Warsaw");
    Set<String> set1 = Sets.newHashSet(list1);

    assertEquals(3, set1.size());
