Floating Tables and Images

In CSS / HTML

While building up my website I've learned a thing or two about floating tables and images. Floating lets you put text and images alongside of each other, letting you use the space on your page more wisely. Used well, it can also get your message across more effectively.

The code

Here are the CSS definitions for the floating table and images above as well as the captions that you see beneath them.

Listing 1: CSS definition for a floating object and its caption

.floatedObject  {
    float: right;
    caption-side:bottom;
    margin-left: 40px;
    margin-bottom: 10px;
}

.floatedObject table {
    padding: 0px;
    border: 1px solid black;
}

.precaption {
    line-height: 1;
    font-size: 10pt;
    font-family: 'FontAwesome';
    font-weight: 600;
    margin-bottom: 0px;
    padding-bottom: 0px;
}

Here's how you use these definitions in the HTML. The only tricky thing is that, in the HTML, the table or image comes before the text that you want to pair it with. What is really floating, then, is the text--not the table--in the sense that the text "floats" upward until it's level with the table.

Listing 2: Using the floating object styles in HTML

<div class="floatedObject">
    <img src='./image/DeathAsConfessor.jpg' alt='missing' width="250" height="185"/>
    <div class="figcaption">
        <p>Death and the Knight in <em>The Seventh Seal</em></p>
    </div>
</div>

<p>And this paragraph is paired with a floating image. The styling is the same as for the floating table above. I'm adding another sentence here to make the paragraph a bit longer. And another here. And one more sentence for good measure.</p>

The above example shows how to float an image. Floating a table will look exactly the same, except you'll replace the one line starting with "img" with the several lines defining your table. If your table has too many columns, or the contents of the columns are too long, you'll want to use a traditional table. An example is the very first table at the top of this page. Here are the CSS style and the HTML that produced that.

Listing 3: CSS and HTML for a traditional table

CSS:
table   {
    padding: 0px;
    border: 1px solid black;
}

th, td  {
    padding: 4px;
    border: 1px solid black;
}

HTML:
<table>
	<tr>
		<th>Title</th>
		<th>Director</th>
		<th>Country</th>
	</tr>
	<tr>
		<td>Seventh Seal</td>
		<td>Ingmar Bergman</td>
		<td>Sweden</td>
	</tr>
	<tr>
		<td>Seven Samari</td>
		<td>Akira Kurosawa</td>
		<td>Japan</td>
	</tr>
	<tr>
		<td>Seven</td>
		<td>Andy Sidaris</td>
		<td>United States</td>
	</tr>
</table> 
<div class="figcaption">
	<p>This table is styled normally.</p>
</div>

But what if you DON't want to float?

If you don't want text alongside your table, simply don't float the table--use the normal table html given above. But suppose you have two small tables, or two thumbnail images, and you want them to appear side-by-side? In this case you would float them both. The problem is that, if you float the two tables or images, then any text that follows them will be placed next to them.

Here is an illustration of the problem I'm describing.

The Dance of Death.

At the chessboard.

Here's some text I DON'T want to appear alongside the objects which, much to my annoyance, are floating right next to it.

To get around this, you need to clear the floating style. A simple way to do this is to use a CSS style for the paragraph which you don't want to float.

Here's the very simple CSS definition:

Listing 4: CSS for a non-floating paragraph

.nonFloatingPar	p	{
	clear: both;
}

And here's the HTML that makes use of the definition:

Listing 5: Using the non-floating paragraph style in HTML

<div class="nonFloatingPar">
	<p>Here's some text I DON'T want to appear alongside the tables.</p>
</div>

Use of the HTML above will stop not just the paragraph itself from floating up alongside an image or a table. It turns off all floating altogether.

Which leads us to a bit of a kluge. Sometimes you want to stop all floating of any elements below a certain point on the page, but you don't have anything to say within a set of <p> </p> tags. The snippet below makes use of the nonFloatingPar style with an empty paragraph.

Listing 6: Using the non-floating paragraph style to stop floating from this point on the page, down.

<div class="nonFloatingPar">
	<p></p>
</div>
